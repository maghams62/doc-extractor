Quickstart:
1) Backend deps
   cd app/backend
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   python -m playwright install
2) System deps (macOS)
   brew install tesseract poppler
3) Frontend deps
   cd app/frontend
   npm install
4) Run
   Backend: cd app/backend && PYTHONPATH=.. uvicorn backend.main:app --reload
   Frontend: cd app/frontend && npm run dev

Artifacts:
- Extraction/Autofill runs are stored at `app/backend/runs/<run_id>/`
  - inputs/
  - extracted.json
  - run.log
  - trace.zip
